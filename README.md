# ubuntu-1404-vagrant

VM for ruby and node dev

# Generic Steps

* Install VirtualBox
* Install Vagrant
* Run run.sh
* Enjoy

# OSX Steps

```
brew install caskroom/cask/brew-cask
brew cask install virtualbox
brew cask install vagrant
git clone git@github.com:joejag/ubuntu-1404-vagrant.git
./run.sh
```
